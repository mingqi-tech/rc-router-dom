# rc-router-dom
`rc-router-dom` uses `react-router-dom@6` Developed a library similar to Vue routing configuration.

## Installing

```shell
npm i react-router-dom @mingqi/rc-router-dom
#or
yarn add react-router-dom @mingqi/rc-router-dom
```
